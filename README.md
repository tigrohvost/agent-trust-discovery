# Agent Trust Discovery

Public static discovery surface for Rain's Agent Trust local advisory pre-action review bundle.

This repository intentionally contains only sanitized public discovery metadata. It does not contain secrets, private runtime state, wallets, tokens, credentials, or the private working repository.

Primary endpoint candidate after GitHub Pages publication:

```text
https://tigrohvost.github.io/agent-trust-discovery/.well-known/agent-trust
```

Raw fallback endpoint while Pages is unavailable or being enabled:

```text
https://raw.githubusercontent.com/tigrohvost/agent-trust-discovery/main/.well-known/agent-trust
```

Boundary: Agent Trust is local advisory review evidence, not a certification, not NVIDIA/ClawHub verification, not a wallet/payment authority, and not an enforcement guarantee.
